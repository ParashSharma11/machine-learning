# Sales Forecasting – Walmart Dataset

## Business Problem
Retail companies need to forecast future sales to manage inventory,
staffing, and promotions efficiently.

## Dataset
Walmart Sales Forecasting dataset from Kaggle.

## Objective
Build a machine learning model to predict weekly sales using historical data.

## Approach
- Exploratory Data Analysis (EDA)
- Feature engineering from dates
- Regression models for prediction
- Model evaluation using R² and RMSE
